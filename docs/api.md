# Kerala Live API

OpenAPI is generated by FastAPI at `/docs` and `/openapi.json`.

## Public Endpoints

- `GET /api/districts` returns the homepage district grid.
- `GET /api/districts/{district_slug}/alerts` returns active official alerts.
- `GET /api/districts/{district_slug}/reports?date_filter=today&sort=newest` returns the district feed.
- `POST /api/districts/{district_slug}/reports` creates a community report.
- `GET /api/reports/{report_id}` returns report detail modal data.
- `POST /api/reports/{report_id}/comments` adds a comment.
- `POST /api/reports/{report_id}/verifications` records confirm, incorrect, or resolved.
- `POST /api/reports/{report_id}/images` attaches an optional report image.

## Admin Endpoints

- `POST /api/admin/districts` creates a district.
- `PATCH /api/admin/reports/{report_id}` approves, rejects, or changes report status.
- `PATCH /api/admin/comments/{comment_id}` approves or hides comments.
- `POST /api/admin/alerts` creates official alerts.

There are no search APIs, pagination APIs, user profile APIs, or authentication systems in this scaffold.
